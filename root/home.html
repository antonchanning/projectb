About Blockpress

Blockpress is a front end web program, written in java script and jquery. It allows the user to create highly customisable and themable websites, that can make use of ajax loading content modules to pull data from multiple javascript apis, including steem, decent and bitshares.

Steem modules

  • Stream of posts by username and tag.
  • Steem post by post id
  • Steem posting form

Bitshares modules

  • Price graph for specified crypto currencies.
  • Price comparison between two crypto currencies
  • Top crypto table by specified metrics

Decent modules

  • Latest content to buy from user.

Examples uses

  • Making multiple websites streaming content from a single steem user and using selected group of tags.
    You can post everything using one login on steemit.com and use tags to create separate blog sites for your personal and professional content (or anything you are passionate about!)
  • Making a single website streaming content from a multiple users and select group of tags.
    If you happen to have multiple accounts for your art or programming, you can use Blockpress to stream them together in one blog. This way you can showcase all your work in one place.
  • Creating a forum website that streams all content from single or multiple tags to create forum categories.
  • Creating a moderated forum website that streams all content from single or multiple tags but a select group of users.
  • If a user spams a tag, you can remove them from your Blockpress stream. This gives you a quality control over the content displayed on your website.

  • Selling decent based content on your steem based media blog.

  • Displaying relevant crypto charts on your crypto themed blog.

  • Displaying relevant crypto charts on a private VNP based site.
